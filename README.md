# T-TRACKER · ACF — Final Version

## Deploy on Replit (3 steps)

1. Create a new **Python** Repl on replit.com
2. Upload ALL files — keep `static/` folder with `index.html` + `ACF_logo.png` inside
3. Shell: `pip install -r requirements.txt` → click **Run**

## Publish publicly
After running: click **Deploy** (top-right in Replit) → Autoscale → your public URL is live.

## Features
- Arabic/English toggle (AR/EN buttons in header)
- Dark/Light mode toggle (moon icon in header)
- LIVE blinking badge on every card
- Real ACF logo (green leaves + blue drop) in sidebar
- Channel filter pills (All / per channel)
- Branded share buttons: WhatsApp (green), Telegram (blue), SMS (purple)
- Auto-scrape every 30 seconds with bell notification
- Bookmark, keyword highlight/filter, Excel export
